# 🎡 大转盘前端网站 — 设计文档

## 一、项目概述

一个「活泼可爱」风格的幸运大转盘交互式网页，通过 Node.js 提供前端静态服务。

## 二、视觉风格

| 属性 | 设定 |
|------|------|
| 风格 | 活泼可爱、多巴胺配色 |
| 主色调 | 粉 #FF6B9D、橙 #FF9F43、黄 #FECA57、绿 #48DBFB、紫 #A29BFE |
| 背景 | 浅渐变，柔和气泡/星星点缀 |
| 字体 | 圆润无衬线 (系统默认) |
| 圆角 | 20px+，布丁感 |
| 动效 | 弹跳、闪烁、彩色纸屑 |

## 三、功能模块

### 3.1 转盘主体
- 8 个彩色扇区，每个扇区带 emoji + 奖品名称
- 旋转动画：CSS transform rotate + transition 缓出
- 指针：固定在顶部的三角/爱心形状

### 3.2 交互控制
- 中央大按钮：「🎯 抽一次」
- 点击后转盘加速旋转 → 减速停止 → 弹窗显示结果
- 每次旋转结束后可再次点击

### 3.3 结果弹窗
- 全屏半透明遮罩 + 圆角卡片
- 显示中奖 emoji + 奖品名称
- 底部「再抽一次」按钮
- 伴随彩色纸屑 (confetti) 飘落

### 3.4 音效 (可选)
- 旋转时"嗒嗒嗒"模拟
- 中奖时"叮"提示音

## 四、技术栈

- **服务端**: Node.js + Express
- **前端**: 原生 HTML + CSS + JavaScript (零依赖)
- **动画**: CSS transition + requestAnimationFrame
- **纸屑**: 原生 Canvas confetti

## 五、文件结构

```
lucky-wheel/
├── server.js          # Node.js Express 服务
├── package.json       # 项目配置
└── public/
    ├── index.html     # 主页面
    ├── style.css      # 样式文件
    └── script.js      # 交互逻辑
```

## 六、启动方式

```bash
cd lucky-wheel
npm install
node server.js
# 访问 http://localhost:3000
```
